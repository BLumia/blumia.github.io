BLumia's Dairy Wrapzone

Is that BLumia's Darkness World before?

June 5, 2017

git reset --hard v2013

说起来我大概有一整个学期的时间都在做调整心态这一件事吧，而因为这个问题我一直实在不知道如何处理，我状态变差似乎变得愈演愈烈停不下来了。大三下学期末我们就都被要求除考研学生外别的学生不得住校，于是我不得不选择把我的宿舍的所有东西都清空，一些搬不走的东西就等着被丢掉，却无意间漏收拾了我高中时期的日记本，好在我同学注意到了那个本子并帮我收拾起来了，我也在上周回校拿回了那个本子。昨晚心态依然不好，想到拿回了日记本，就开始翻看我之前写的东西，翻着翻着，我好像发现，从那时候到现在，我好像遗失了一些东西…

奈伊呜和付琴 美相统

虽然高中也是有那么多烦恼，但现在看来，我的每个烦恼都是一些单纯而简单的事情。虽然现在我看上去每个阻碍我的烦恼看上去都相互关联且复杂的像是ubuntu里的依赖关系，但如果仔细去想，我为何要去困惑那些烦恼背后的关联呢？固然，追寻源头可以帮助我更好的解决问题，而当一切加起来超过我能够承受的threshold后，我便不可能再能够处理的好他们了，也只能任由它们扩大着而无从开始下手解决。

或许我应该变得天真一些，不再总是关注那些烦人的事情背后到底是什么联系，而是选择单纯的一个个去解决他们。总是看着一个又一个烦恼堆积到一起，显着自己愈加渺小和肤浅，但把他们关联起来的人，正是我自己啊。

一个天真的人很奈伊呜，一个肤浅的人很肤浅。

重回敬重

我本身一贯厌烦自视高傲的人，也很厌烦嘲讽，却不知什么时候开始不知不觉地嘲讽别人了。有时候看到别人一点错就开始觉得这个人可能只是在装懂就上去指正一般的说道，有时候又似乎是为了纯粹的证明些什么就上去一争高下。

回到高中时期，无论怎样都尊称他人为您，无论如何都遵重的看待各种各样的事物。我一直信奉的一个标杆是“不能理解的事情就不要去理解”，或者叫求同存异，我也一直认为，每个人都有他优良的一面。而我却不知什么时候，愈加变得轻浮而急躁且毫无理由的高傲了起来。

我的知识水平固然不及很多人，我目前所活跃的两个群里我大概都能垫底吧，而无论哪里哪个群哪些人，我都应该回到像高中那样，保持那份敬重，少一分浮躁吧。

社会层次格局

无论是有意还是无意的，我都一直在思想上反对社会层次格局啊阶级啊之类的论调，我也一直认为，实力就是实力，什么贫富啊什么家庭背景啊人脉啊都不会对生活有什么决定性的影响。但也是大学后，我才渐渐开始觉得，我们所接触的所有东西，也都是由环境直接或者间接影响的，比如小学就开始接触计算机和我这高中才算起步的差距，或是一些别的层面的思考方式之类。

回想四年前，我虽然反对这种分化的论调，却也潜意识的接受着现状但不去想它。我依然申请着助学金却在申请递交之后无论怎样都不再让此事干预我本应该去做的事情。而想到现在，我每做一件事，想到由于总总格局差异带来的隔阂或者阻碍，就让我畏于继续。

我现在依然更相信实力才是改变现状的方式，却又开始逐渐试图接受层次格局的“事实”，这都没什么。无论怎么想，我都不应该被这些隔阂或者阻碍影响到我本身的想法和行动吧。

接受到底有多难

一些思想上的抗争，其实本身也是由自己带来的。有些事情即便事实摆在眼前我也很难接受，有时接受一些事情的同时仿佛就立即推翻了自己一些思想的基石。不过，如果回到高中时期的我，大概会认为，事实如此又怎样呢？倘若无所顾忌的去欣然接受一些事情，不反而更能让自己安然的做好下一件事吗？

顺从内心

顺从内心大概是我的所有想法的一大基石吧，但我却发现，一旦需要顾忌的事情越来越多，就愈加难以顺从内心去做事。如上几条，我都没有举什么例子去说明我为什么会遇到那些问题以及我为什么觉得我应该那样做才对，也是因为顾忌具体的示例可能会带来坏处吧。于是我还是没法在本来目的就是顺从内心的日记上写出真的完整的从心而发的话。

这篇我已经努力的在克服一些问题了，虽然这篇除了我自己之外大概别人看到会觉得逻辑混乱而不知道在讲些什么，但我觉得这算是我给我自己的一个标志吧，我可能错误的做了一些违背初衷的事情，可能遗失掉了一些东西，我觉得我能从高中时的我身上重新找到一些东西，这篇希望可以成为一个新起点吧。

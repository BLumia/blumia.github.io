即将回到 ghpages ?

由于 ghpages 不支持 https 以及一些别的原因，前些时间我把博客从 ghpages 迁移到了 别的地方 。这几天 GitHub 发布了新闻称 ghpages 终于支持 https 了 ，而为了降低维护成本（空闲时间的确不多，也能省点钱租服务器），大概停一段时间我的博客会再次迁移回来吧。

由于我当时迁移出去的时候做的简单博客系统脚手架本身就是为了省事儿而直接对 hexo 的 markdown 文章（甚至 front-matter）做了完整支持的，故实际迁移并不是很麻烦，不过由于我的确不熟悉+某种程度上不喜欢 hexo 这一套的工作机制，或许我会换用 jekyll 吧。当然，还有一个问题是 ghpages 好像只能设置一个域名的解析，如果多个的话我似乎还是得自己找个服务器做处理，所以我那个造轮子博客系统脚手架或许还会接着用…吧？当然，不用 hexo 这一套的另一个原因是一些页面的确需要更新或者直接移除了…不过这些都是后话。

公开的牢骚也很久没发了，少发点的好，不过最近还是对很多事情有了新的认知和看法，说不定哪天又能更新一篇，到时候再说吧。

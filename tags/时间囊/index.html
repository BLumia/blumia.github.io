2017年末小结

今年的时间囊开囊大概特别惨，于是这次开囊之外还会带一些别的东西，也回归到 “年末小结” 的主题。

时间囊开囊

按照去年（或者说，没过完的今年）的时间囊格式，大概是每个大标题对应一件或者一组事情，下面依然按照这种格式来总结。

前端特技化工具链

去年写到，我意向上大概是不打算从事前端相关的工作的，于是尽管一直计划要学点前端特技，结果还是没去主动学。事实上，这一年我好像也可以说是 “挺忙的” 了，以至于好像很多东西都没能抽出时间学，至于我忙什么了这个问题，得放在下面说。

如果一定要凑个数，我可能还算学了一点东西的。ZUN 的 东方天空璋 发布后，由于一贯的习惯循环正作的 BGM 的需求，于是改造了 chris 的 cli 版 thplayer （好像没找到链接），用 Qt 搞了个 thplayer （虽然后来的代码也都是 chris 写的）。在基本功能齐了之后我给这个 thplayer 写了个主页，用了个叫 moon.js 的东西，一个轻量级的 MVVM 库，当然，使用方式和 vue.js 还是有很大区别的，于是其实还是算啥也没学。

前端工具链其实给我一种比较奇怪的感觉，而进来突然火热的 parcel 则好像是一次针对我认为存在的前端工具链现状问题的修正的尝试。具体我不大好评价（所以我也没说这种奇怪的感觉到底是啥），但估计要真想去前端娱乐圈探探水深，还是得凑出一些时间好好研究一下这些东西。

C++从入门到放弃

于是必须得重新拾一些基本功以后面试才有底气，于是得重新啃一下C艹。

嗯，目的是给面试留底气，结果我目前参与过的几次面试也都没问我 C++ 。

我拿到目前 L 司[1^]实习后，实习期的项目压力均比想象的要大一些。由于怕对 C++ 的不熟悉导致无法按时完成任务，我的项目基本都以纯 C 或者 C 味 C++ 编写，于是导致我还是没成功复习我那忘干净的 C++ 。当然，复习 C++ 大概是迟早的事情，即便这年没能复习成功，2018也早晚要好好复习一下的吧。

一门生产力语言

L 司实习从一开始就可以说是荒诞，我拿到的 offer 是 Linux C 部门，结果（转正后）实际工作用的却是 lajilava 。因为我对 lava 的一贯 0 好感，我在这一年也没开始对 lava 的系统的学习（目前：xjb写），尽管项目代码的阅读也多少让我学到了点东西。

实习期倒是没有接触那套东西，于是实习前的需要 GUI 的项目我选了用 Qt 来搞。Qt 这东西我也算是不陌生了吧，而当时本来也打算趁机学一下 QtQuick ，结果可能是项目紧张带来的压力，我花了一天的工作时间也没搞明白 QtQuick 的玩法，结果项目还是用了 QWidgets 而不是 QtQuick ，当然，QtWidgets 倒是更熟悉了一些。

C# 的确没怎么学，买了书，没时间看。Electron 也同样没抽出时间玩，于是…如果上面那个“更加熟悉 QtWidgets” 不算的话，那我这个也算没完成了。

大概不会找对象

我比较怕前几天撩我的人看到这篇，不过还是写一写吧。

在比较早的时候我检查学校 OJ （今天zznuoj宕机了吗？）的时候，发现我神奇的有一个站内信，有个学妹（当时以为是学弟）问了我 QQ 号，我以为是想接手 OJ 的人于是就给了 QQ 号，后来那人就加了我， QQ 设置性别男，名字听起来也是男生的。

后来那人第一次问我，得知不是接手 OJ 的，又说能不能辅导数据结构，我以为又是一个懒得写作业的人，就按以往的做法回复了（大概类似 “搜索引擎是你的好朋友” 这种），再后来第二次问我，则是问了一个奇怪的问题。

“假如你有一个关系不错的异性朋友 \n 她突然在某一天不明不白的对你翻脸不认人，进而在聊天软件上对你说很难听的话，你会是什么感受？”

嗯，我依然认为ta是个男生，于是我开始觉得事情有些奇怪，并在 bkt 吐槽说“问这个我哪知道，我又不懂这个”，并且也这么回复了那个人，当然，万恶的 QQ 导致我那条回复压根没发出去，而我那时候也没反应过来，以为我发出去了。

后来感情专家 jjy 分析说这是个妹子想试一试看我合不合适…这怎么可能？于是我也没大当回事，并告诉jjy这人是男的别瞎想。后来 jjy 建议我说 “被盗号了”，我告诉jjy我已经回过了，说我不知道，结果许久没反应，下意识点了一下历史纪录才发现我压根没发出去消息。于是我按着jjy的建议回说 “被盗号了？” ，结果得知说是那人的亲身经历，再后来她表示她是女生。

再后来对方表示这是一年前的事情了，而后来对方的话被 jjy 完美猜中，我才开始觉得我是不是真的被撩了。当然，后来的一句话让我突然觉得很有趣：“你是不是叫LC(我前acm队友)？”

哦…冲着lc去的么，结果错误的找到了我么…

不过第二天她发来了一些看上去好像更像是表白的话，由于我的确觉得这事情太突然了，于是我选择装傻当没看懂…

jjy 说，主动的女生太难找了，但是我觉得，一个既没见过，也没听说过的人，之前也不了解，也不知道性格是否近似，连面也都没见过，既然如此就打算要试一试，是不是有些…太草率了？如果你是这位女生，有意或者无意的看到了这一段，我希望你能理解，太突然的出现在我生活中，措手不及一般，毫无征兆，互不了解就要恋爱的话，我的确不能接受。或者说，至少目前不能。

年末小结

到了总结今年发生了什么比较重要/大/有意思的事情的时候了，依然分节写吧。

很不巧的两次生病

今年实际生病次数是2还是3来着，但至少其中两次生病极为不巧。

第一次生病是我秋招决定投递简历后[2^]，拿到了腾讯的面试邀约，结果去（西安）面试那天前重感冒+发烧了，到了西安依然不见好转，结果迷糊了一上午，下午状态也不佳，结果面试比较惨。尽管当时觉得二面还有戏，结果还是挂了，次日滚回郑州感冒依然没好。

第二次则更蛋疼，前一段时间，冬天，由于 LC 感冒了，他认为是寝室太干导致的，于是晚上决定不开空调加温，结果我就冻感冒了，然后就赶上 CET 6 了。实际上这次 CET 6 是我第一次参加 CET 6 [3^]，结果我感觉是过不了了。

对于这两次不巧，我也没什么办法，不过我也应当从生病之外找一下原因。腾讯面试那次，对于我写的项目经历，我写了在 L 司实习时写的分布式网盘，结果在面试官得知项目并没有实际上线后整个面试过程都变得微妙了。实际上，实习期这个项目的确是练手项目，我认为练手项目不上线是很正常的，而后来一段时间看我那些去北京培训机构的同学，我才反应过来，对方是不是把我的项目当成是我瞎编的了吧？会不会把我当成是假货了？当然，我的过错就是，白纸写代码的链表排序直接是冒泡了（我一开始还愣了一会儿），而后续问了我一些我平时边 man 边写的东西的细节，我说我不大记得了但是可以 man 出来，这也可能导致对方认为那个东西不是我自己写的——于是总的来说，面试还是得背一些东西的吧，比如快排，又比如常见函数调用到底怎么用——尽管我依然觉得这很没意义。

CET 6 则大体怪我，我的确没抽出时间准备，底子也并没有那么好，考不过也没什么可说的。不过有比较有意思的事情，卷子上的翻译题，就是 Wikipedia 上 Dongting Lake 的第一段原文翻译成中文，然后题目让翻译成英文…

今年看了很多次电影

由于同学是漫威粉，我就被拉着去看了好几次电影，不过大都没什么内涵。此外，年初《降临》上映，我和我爸还是去影院看了，尽管有其要表述的内容，但我觉得这部影片并不能让我满意。

我不太记得我把“今年看了很多次电影”这条写在备忘是我想写点啥了，先放着。

经济独立？

比特币的火爆，我身边的人也有开始玩的了，同在 L 司的学长买比特币后来的确涨了一些，不过央行发文后跌了一波，由于以后就可能无法交易结果他全抛了，当然，结果是亏了，不过他现在对于区块链技术的虚拟货币依然很感兴趣。

当然，我是觉得我是玩不动虚拟货币的了，但是这促使我去了解了经济独立这个概念。如果达到了，那我就可以不必担心收入的去做我喜欢的事情了，也是一个很理想的状态，当然眼下对我来说这不可能。

不过在这段时间，我的确开始对创业感兴趣了，一方面是这可以做一些我认为有意义的事情，另一方面我也不必担心失去工作。然而若要创业，必然得有资本吧，无论是作为大头的经济资本还是技术资本。于是说不定，在我觉得有机会的时候，试试创业？

本心论

大概是工作压力导致，我所谓“哲思”的时间变多了，虽然大都是消极的，但都基本围绕一个议题，即我在做的事情是不是我真的想做的事情，或者说，我违背本心了吗？

上面提到的创业，其实也是我这些想法导致的。实际上我对我现在的工作并不满意，虽然公司许诺的福利没有达到以至于让我觉得丧失信任感，以及试试才知道很不爽的单双休这两点就很值得吐槽，但这还不至于让我觉得我应该离开。而我觉得我不喜欢这里的原因大概是，要做的事情本就不是我喜欢的，而这也不是我想要的生活吧。

现实一点说，如果没了工作，我照样没办法做我喜欢的事情（“没有钱是万万不能的”），而我也不能保证有下家后就一定比现在好。头条据说加班到晚上九点呢，tx游戏部可是很肝的，对吧？

作为可能称得上我思想根基的观点的本心论（暂且这么叫），我在其它事情上也尽可能保证遵循这点，不恰当的例子的话，比如我开始拒绝玩 osu! 而去玩 o!b 之类。

当然更详细的内容还是不写在这种“公开”一般的日记里比较好。

愈加消极的 bkt 群？引导？

事实上意识到这件事已经很久了，但是我却开始不知道我该怎么做。尽管抑郁症一词被滥用导致我一度很反感这个词，但当实际发现似乎有人的确患抑郁症时，我除了劝导就医之外却完全不知道怎么做。

我认为环境的确是可以影响人的，而愈加消极的 bkt 或许可能导致更多人出现心理疾病。——对找工作一度失去信心的 o 变得愈加消极颓废；MtF开始吃糖等等。我一度开始拿着“不能理解的事情就不要去理解”这一点来告诉自己我可能只是不能理解他们所以我不需要去管，而就拿MtF吃糖而言，当得知那些副作用后，我却觉得，似乎我的确应该做些什么。

不过我认为最大的问题在于，我即便到现在也没能调整过来我的心态问题（尽管现在似乎好多了），那我又应该怎样的引导呢？我到底应不应该做呢？

TimeCapsule - 2017

于是现在该写17年的时间囊辣

鉴于开去年“时间囊”的时候都忘记上次时间囊写的时候我在想些什么了，于是这次干脆分条带原因写得了。于是下面是内容。

前端特技化工具链

首先想到的是打算系统的学一下前端科技，前端工程化的那堆东西，node.js啊webpack啊babel啊ES6啊什么的。原因两个，一个是jxjxj学这个的确吸引到我了（看着就好厉害），另一个就是翻github repo的时候会看到大家都在用，也就我还在写那种一点也不优雅的意大利面htmlcssjs代码。

当然，如果一定要说还有别的原因的话，那就是因为班里有一些人开始啃html+css+js了，学新的是为了能再度把距离甩远。至于以后会不会从事前端方面的工作，不敢断言，但是我意向上不想做这个..

顺便，这个似乎不算纯零基础，之前的野路子还是用过点Node.js的

C++从入门到放弃

这是要把复习c和c艹提上日程了。因为距离毕业愈加接近了，我觉得我大三才意识到刷绩点的重要性我觉得我之前扔掉那么多，考研基本无望了，于是必须得重新拾一些基本功以后面试才有底气，于是得重新啃一下C艹。

至于为什么是C艹，首先我有一些脑洞大概会选C艹去实现，比如之前写了一半的PineappleSynth的效率问题还没来得及填坑，以及没准我还真会去写个Midi Sequencer？别的的话，我还有个坑是实现一个跨平台的简单图形api，这个没点底子我估计我连个wrapper都写不好。哦对，我现在连个非托管式的dll都没写过…

2015年末小结

转眼就到了年末，是时候小结一下今年的得失了。也换一种方法来开所谓的时间囊。

于是当我打开15年时间囊的页面的时候，我发现我说“九木集性质的文章不会再发上来了”，然而我还是发上来一些…不过还好…该在硬盘呆着的还在硬盘呆着呢..这种杂集文章发出来自己会感觉好一些，以后翻过来还能看看自己之前干过的傻事儿。至于别人看到不看到，其实本来就没打算让很多人看到，能到这里的应该都是关系非常好的朋友了吧。所以看到了应该也没什么吧。。不过，或许真的有一天我会把我记水事儿的文章统统收回硬盘然后只留个什么所谓的技术博客？

接着看时间囊，那么下面也分块写。

1. Adobe常见软件的使用


老实说，我Ps用的更熟练了，虽然依然不会玩什么高级花样。总体对我来说，这些软件并不是我的硬性需求，所以我也没怎么专门去学/瞎搞。不过说起来，我现在主力系统换成linux后，编辑图片的时候还是会“想念”Ps。。。哦对了，Ae和Pr的workflow大致了解了，也大体会做一些简单的操作了，大致和我会Ps的程度一致…
ps. Vegas比Pr好使啊..

## 2. 其他常见软件使用


前俩实际都是指Git..由于目前我还是经常会选择给我的乱搞的小东西加上vcs，所以git的很多操作还是会的。不过命令倒是不会多少，也就能commit和push的程度（
无论是主力系统换成linux，亦或是别人安利，Vi/Vim我多少会了一些基本操作，不过这些操作都很简单，只是navigation的那部分。高级的…至少目前没怎么玩过。
水果（fl）的workflow大体也熟悉了，不再像之前一样只会在能看到的地方乱拖乐器瞎搞了。Ableton Live这个安利的新软件我还不怎么会玩，行走蛋糕依旧只会拿来当个midi编辑器用，哦对了。。圣诞特惠（现在依然在持续其实..）行走蛋糕在sbeam只要一百多，当时看到的时候真的想买。。。
Unity3D这个坑比较有意思，本来打算明年（大三/大二下半学期）再开坑，结果某一天上网的时候突然发现unity3d出linux版了（这时主力系统已经转linux了），异常兴奋，于是下载了，并开始慢慢了解unity5的各种姿势。不过。。linux上写c#真不爽啊。。

## 3. 新的编程语言


Java我有课所以不管这坑开不开都是要学的。不过尽管有课但是我依然没在平时写过java代码倒是真的。。C#则写了一部分。因为上面提到的Unity3d的坑开了的缘故才写的。但，java和c#根本没有网上说的百分之多少的相似度好伐？M$有好多黑科技语法糖什么的呢…至于Lua，这坑压根没开，Py反而开了但也没学多少（只是看了语法规则。。），相反倒是正经学了学node.js…

## 4. 旧知识的复习


自己很久都没搞什么自己的小项目了，所以我觉得我也没怎么复习cpp和oop。虽然好歹有个思想，学java的时候也复习了一部分。数据结构的话，有课，所以必须学，不过还是那么些东西。至于算法，或许忘得更多吧，不过这个毕竟会用上，会去复习并学些新的算法的。

## 5. 工程和项目化


不得不说HappyRoll这个让我可以说“爱过”的MarbleBlast Mod我停工了，有一次有人提醒我说论坛那边问是不是dead了，于是我过去发了个目前suspend了。的确，我觉得学点东西之后才能搞出更好的mod，但是实际上我不太想继续这个mod了，或许应该换个引擎，搞个新的，免费的，awesome的滚球游戏出来。说到底，滚球游戏才是能让我答“爱过”的东西。额，我想表达点什么东西但是不知道该怎么写出来了。想到了的话再补吧，或者以专题写出来。
额..U3D滚球这个在上一自然段描述过了…
至于ToK这个，其实我没兴趣，不过依然值得一写。说实话这是我第一次见识到什么叫“就差一个程序员”系列，以及目前的所谓创业圈到底多奇怪。（个人声称是公司并）拿着一个只有游戏流程设计的构思然后说着“我们正在搞”就可以拿到奖，我已经猜到创业圈有多奇葩了。出去参加创业比赛可以和名人谈笑风生，的确，但是又有什么用呢？[1]
安卓开发没入坑，放到明年了。ios算了，没设备。至于wxWidget。。我反而去看了gtk…


## 6. 网站开发部分


hustoj前端重写，实际上也就是BLOJ了，因为是完全重写所以并不是hustoj的什么fork。不过目前我没动力去写这个了。。因为我发现似乎根本没人会用。。至于贴吧仿制app我只是拿php搞过一个相当粗糙的版本。目前不打算搞了，我觉得即便要搞也得上mvc或者用一些科学的办法吧。另外就是。。我觉得php太蛋疼了。。
个人软件发布app我目前没有这个需求了。所以不再打算搞，不过我有个搞社交化软件/资源共享平台的小脑洞，这个新坑留着。至于个性化个人博客，我觉得我做的过头了，这一年都从wp到typecho再到hexo了。。。。

## 7. 其他内容


日语这个我本来报了下一学期（大二下）的选修，然而没选上。至于乐理，本来以为上音乐鉴赏会讲一些，谁知道只是讲了讲五线谱就完了。不过下学期我报了电子钢琴。。至于绘画。我觉得这坑开不了，因为连个基本的路线都没有。无从下手吧。

至于本年总结性质的东西这次就不写了。
